# Gates of Olympus Playable Demo

## Структура проєкту

- `index.html` — Головний файл, підключає всі стилі і скрипти.
- `/css/main.css` — Базова стилізація (адаптив, layout, кнопки).
- `/js/` — Вся логіка: анімації, рендер, героя, символи, звуки.
    - `main.js`: старт і рендер екранів.
    - `screens.js`: логіка перемикання екранів, рендер інтерфейсу.
    - `animations.js`: зміна стану Зевса, анімації компонентів.
    - `sound.js`: відтворення звуків.
    - `symbols.js`: конфіг слоту, комбінування символів.
    - `utils.js`: утиліти, локалізація текстів.

## Особливості
- HTML5, чистий JS, CSS Grid/Flex (без абсолюта)
- Адаптивність під 9:16
- Локалізовані тексти
- Плавна анімація на кожному екрані
- Легка заміна медіа-файлів та звуків

## Як запускати

1. Копіюйте структуру.
2. Додаєте ресурси у assets.
3. Відкриваєте index.html.
