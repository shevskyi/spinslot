# Gates of Olympus — Playable Demo

## Project structure

- `index.html` — Main file that includes styles and scripts.
- `/css/main.css` — Base styling (responsive, layout, buttons).
- `/js/` — All logic: animations, rendering, symbols, and sounds.
    - `main.js`: app entry and screen rendering.
    - `screens.js`: screen switching and UI rendering logic.
    - `animations.js`: Zeus state changes and component animations.
    - `sound.js`: sound playback utility.
    - `symbols.js`: slot configuration and symbol definitions.
    - `utils.js`: utility functions and localization helpers.

## Features

- HTML5, plain JavaScript, CSS Grid/Flex
- Responsive layout targeting a 9:16 ratio
- Localized text support
- Smooth animations across screens
- Easy media and sound asset replacement

## How to run

1. Preserve the project structure.
2. Add assets into the `assets/` folder.
3. Open `index.html` in your browser.
